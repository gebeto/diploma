Основною вимогою до запуску проекту є наявність встановленого docker та docker-compose на компютері.

Запуск застосунку:
1. Перейдіть у директорію проекту в консолі 'cd application'
2. Запустіть сервер скриптом './scripts/production-start.sh'
3. Зачекайте поки проект збереться та запуститься
4. Відкрийте у браузері сторінку 'http://localhost:8000' або 'http://127.0.0.1:8000'
