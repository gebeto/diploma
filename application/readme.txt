Основною вимогою до запуску проекту є наявність встановленого docker та docker-compose на компютері.

Запуск застосунку на ОС Linux:
1. Перейдіть у директорію проекту з консолі
   - Windows - 'cd application'
   - Linux/MacOS - 'cd application'
2. Запустіть сервер скриптом
   - Windows - './scripts/production-start.bat'
   - Linux/MacOS - './scripts/production-start.sh'
3. Зачекайте поки проект збереться та автоматично відкриється вікно авторизації у вашому браузері.
   - Якщо вікно у браузері не запустилося автоматично то відкрийте у браузері сторінку 'http://localhost:8000' або 'http://127.0.0.1:8000'
4. Введіть у відповідні поля дані для авторизації та натисніть кнопку 'Вхід':
